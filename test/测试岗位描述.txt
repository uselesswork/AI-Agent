星云科技有限公司｜Python 后端开发实习生

工作地点：上海市浦东新区，可每周远程办公 1 天
招聘人数：2 人
实习周期：至少连续实习 4 个月，每周到岗 4 天及以上

岗位介绍

我们正在开发一款面向高校学生的智能求职助手，帮助用户完成简历解析、岗位匹配、投递管理和面试准备。你将加入后端研发小组，参与 AI 应用服务和业务接口的开发。

岗位职责

1. 使用 Python 和 FastAPI 参与后端接口的设计、开发与维护。
2. 参与简历解析、岗位匹配和大模型调用链路的功能开发。
3. 使用 PostgreSQL 完成业务数据建模、查询和基础性能优化。
4. 编写单元测试和接口测试，协助定位并修复线上问题。
5. 参与接口文档、技术方案和项目开发文档的整理。
6. 与前端开发人员协作，完成接口联调和功能验收。

任职要求

1. 计算机科学、软件工程、人工智能或相关专业本科及以上在读。
2. 熟悉 Python 基础语法，理解面向对象编程、异常处理和常用数据结构。
3. 熟悉 FastAPI、Flask 或 Django 中至少一种 Python Web 框架。
4. 掌握 RESTful API 基本设计方法，了解 HTTP 常见状态码。
5. 熟悉 SQL，能够使用 PostgreSQL 或 MySQL 完成基本的数据增删改查。
6. 会使用 Git 进行版本管理，能够在团队协作中完成分支开发。
7. 具备良好的沟通能力、学习能力和问题分析能力。

加分项

1. 有大模型 API、OpenAI 兼容接口、Prompt 工程或 AI Agent 项目经验。
2. 使用过 Docker、Redis、Celery 或消息队列。
3. 熟悉 pytest，具有自动化测试或持续集成实践经验。
4. 有个人技术项目、开源贡献、技术博客或可演示的 GitHub 仓库。
5. 了解向量数据库、RAG、文本嵌入或语义检索。

其他说明

本岗位不要求候选人具备正式实习经历，但需要能够展示至少一个由本人主导或深度参与的软件项目。简历中请注明个人承担的工作、使用的技术以及可以量化的项目结果。
